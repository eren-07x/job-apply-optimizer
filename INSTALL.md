# Job Apply Optimizer — Installation Guide

## Requirements
- Google Chrome browser

## Steps

1. Download and unzip the file
2. Open Chrome and go to: chrome://extensions
3. Turn on **Developer mode** (toggle in top-right corner)
4. Click **Load unpacked**
5. Select the unzipped `job-apply-optimizer` folder
6. The extension appears in your toolbar (click the puzzle icon to pin it)

## How to use
1. Go to any job posting on LinkedIn, Indeed, or Naukri
2. Click the extension icon
3. Create a free account (2 free uses)
4. Paste your resume and click the button
